Click each slot (or drag & drop) to add photos. They'll be renamed automatically for the website.

How to Use This Tool

  1. Open your Facebook page in another tab: Dakota Sky Photos
  2. Right-click a photo → "Save Image As" to your Desktop (any name is fine)
  3. Click the matching slot below and select that saved image
  4. Repeat for all 14 required photos (bonus ones are optional)
  5. Click "Download All Images" at the bottom — you'll get a ZIP with every file correctly named
  6. Unzip into the images/ folder next to your HTML files. Done!
